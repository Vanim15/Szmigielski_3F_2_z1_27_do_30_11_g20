1. W jaki sposób można określić właściwość danego obiektu?

Sposób 1 – notacja kropkowa:
a) dla tekstu:
auto.marka = "Opel Omega";

b) dla liczb:
auto.rok = 1996;
auto.cena = 25000;

Sposób 2 – notacja tablicowa:
auto["cena"] = 25000;
2. W jaki sposób można odwołać się do pól (właściwości) i metod?

Do pól i metod obiektu odwołujemy się przez nazwę obiektu i kropkę.

Przykład:
console.log(samochod1.kolor); – wyświetla kolor obiektu.
3. Wymień cechy charakterystyczne konstruktora
  • jest wywoływany automatycznie przy tworzeniu obiektu,
  • ma taką samą nazwę jak klasa,
  • nie zwraca żadnej wartości,
  • może przyjmować argumenty,
  • do tworzenia obiektu używa się słowa kluczowego new.
4. Konstruowanie obiektu – przykład

var osoba_1 = {
  nazwisko: "Kowalski",
  imie: "Jan",
  zawod: "piekarz",
  wyswietl: function() {
    document.write(this.nazwisko + " " + this.imie);
  }
}
