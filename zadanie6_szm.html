Zagadnienia teoretyczne z JavaScript

1) Rodzaje okienek:

W języku JavaScript występują trzy podstawowe rodzaje okien dialogowych:

  • Okienko confirm – pozwala użytkownikowi potwierdzić lub anulować działanie (zwraca wartość true lub false).
  • Okienko alert – wyświetla komunikat z przyciskiem OK.
  • Okienko prompt – umożliwia wpisanie wartości przez użytkownika, która potem może być użyta w programie.

2) Konwersja liczb:

Wartość odczytana za pomocą funkcji prompt() jest ciągiem znaków. Aby wykonać działania matematyczne, konieczne jest przekształcenie tekstu na liczbę.

Do konwersji stosuje się następujące funkcje:

  • parseInt(zmienna) – zamienia tekst na liczbę całkowitą.
  • toString(zmienna) – zamienia liczbę lub inną wartość na tekst.
  • parseFloat(zmienna) – zamienia tekst na liczbę zmiennoprzecinkową, np. 3.14.

3) Zasady nadawania nazw zmiennym:

Podczas tworzenia zmiennych należy przestrzegać kilku zasad:

  1. Nazwa zmiennej musi zaczynać się od litery, znaku „*” lub „&”. Nie może zaczynać się od cyfry.
  2. Nazwa może zawierać tylko litery (bez polskich znaków), cyfry, „_” oraz „$”.
  3. Nie wolno używać słów zarezerwowanych przez język JavaScript jako nazw zmiennych.

Poprawne nazwy zmiennych: zmienna1, _wynik, $gracz, liczba_test

Błędne nazwy zmiennych: 1zmienna, imię, wynik gry, var

4) Typy zmiennych w JavaScript:

W języku JavaScript typ zmiennej jest ustalany automatycznie po przypisaniu wartości. Podstawowe typy danych to:

  • Number – liczby całkowite i zmiennoprzecinkowe, np. 5, 3.14
  • String – tekst, np. "Ala ma kota"
  • Boolean – wartość logiczna: true lub false
  • Null – pusta wartość
  • Undefined – zmienna niezdefiniowana
  • Object – typ złożony, pozwala przechowywać wiele danych

Przykład:

let imie = "Igor";
let wiek = 14;
let pelnoletni = false;
